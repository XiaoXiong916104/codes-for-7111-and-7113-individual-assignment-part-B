# CHEE7111/7113 Part B Code

This repository contains the MATLAB and Simulink files used for the CHEE7111/7113 Part B individual assignment.

## Folder map

- `Task5_Model_Solution`: nonlinear CSTR model solution.
- `Task6_Calibration_Validation`: calibration, validation and sensitivity analysis.
- `Task7_Simulink_Deployment`: Simulink open-loop deployment and comparison plots.
- `Task8_PI_Controller`: PI controller design and high-setpoint tests.
- `Task9_IMC_Controller`: FOPDT identification and IMC comparison.
- `task10_LTI_MPC`: analytical linearisation and LTI MPC setup.
- `Task11_MPC`: Simulink Linearizer MPC concentration-control case.

Run the relevant parameter script in each task folder before running the corresponding model or plotting script.
